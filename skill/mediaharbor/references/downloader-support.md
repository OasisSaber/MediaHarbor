# Downloader Support

## Primary platform: Windows x64

Linux and macOS paths are best-effort placeholders.

| Tool | Role |
|---|---|
| yt-dlp | Default web probing, VOD, playlists, subtitles, metadata |
| FFmpeg / ffprobe | Merge, convert, validate |
| yutto | Bilibili backup |
| Streamlink | Live streams |
| N_m3u8DL-RE | HLS, DASH, MSS |
| gallery-dl | Social posts, galleries, media collections |
